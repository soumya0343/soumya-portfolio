## ✨ Features

- **Immersive Fluid Art** — WebGL-powered generative fluid canvas animations (yellow & blue themes) rendered via `<iframe>` for GPU-isolated performance
- **Cinematic Scroll Experience** — Multi-section vertical scroll with Intersection Observer–driven animations and staggered reveals
- **Brutalist Typography** — Bold, high-contrast type hierarchy with serif/sans-serif pairings
- **Dynamic Theming** — Dark & light mode support using CSS custom properties with iframe color inversion
- **Fully Responsive** — Adaptive layouts with a collapsible sidebar navigation for seamless mobile & desktop experience
- **Performance Optimized** — Built with Vite + React Compiler for optimal bundle sizing and rapid hydration
